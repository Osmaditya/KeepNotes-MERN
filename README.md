## ✨ Highlights

- 🧱 Full-Stack Application using **MERN Stack (MongoDB, Express, React, Node.js)**
- 📝 Create, Read, Update, Delete (CRUD) Notes with Title & Content
- ⚡ Built RESTful APIs with proper HTTP methods & status codes
- 🔐 Implemented **Rate Limiting using Upstash Redis** (Production-level feature)
- 🌐 CORS handling for frontend-backend communication
- 🎨 Responsive UI built with **React + Tailwind CSS + DaisyUI**
- 🔄 Real-time data fetching using Axios
- 🚀 Deployment Ready (Frontend + Backend separation)
- 📚 Beginner-friendly project with real-world concepts

## 🧪 Environment Variables Setup

### Backend (`/backend`)

```
MONGO_URI=<your_mongo_uri>

UPSTASH_REDIS_REST_URL=<your_redis_rest_url>
UPSTASH_REDIS_REST_TOKEN=<your_redis_rest_token>

NODE_ENV=development
```

## 🔧 Run the Backend

```
cd backend
npm install
npm run dev
```

## 💻 Run the Frontend

```
cd frontend
npm install
npm run dev
```
